configuration
=============

>git config --list
command to list all config details

>git config --global user.name "username"
command for adding a global user.name

>git config --global user.email "useremail"
command for adding a global user.email

>git init
command for initialsing git repository

>git add -A
command for initialising git repository

>git commit -m "commit message"
command for adding staged files to local repository

>git status
command for listing all untracked files

>git remote -v
command to list remote

>git remote add name remote_path
command for adding remote repository to local repository

>git push name
command to push from local to remote repository

>git branch
command to know branch
